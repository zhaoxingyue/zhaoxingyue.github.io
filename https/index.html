科普一下 HTTPS

https 是 secure http，是 HTTP 的升级安全版。 HTTP 是应用层协议，在 HTTP 下面是传输层协议 TCP，TCP 定义数据如何传输，
HTTP 定义数据如何包装。而 HTTPS 是在 HTTP 和 TCP 中间加了一层 SSL/TLS 加密层。
